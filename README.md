# Banking system
Решение учебного задания на отработку навыков ООП на языке C#. 

Код представляет собой консольное приложение .NET.

Задачей было спроектировать базовую архитектуру банковских счетов, используя принципы наследования, инкапсуляции и полиморфизма. Система должна безопасно переводить средства между счетами разных типов, учитывая их уникальные условия (комиссии и бонусы), а также защищать баланс от отрицательных значений при переводах.

## Реализованный функционал
В решении реализовано три вида счетов:
1. **BankAccount (Базовый счет)**: надежное хранение баланса
   * Зачисление средств на счет (`Deposit`)
   * Списание средств (`Withdraw`)
   * Реализация безопасного перевода средств (`Transfer`)
2. **BonusAccount (Бонусный счет)**
   * Наследуется от базового счета
   * Переопределяет метод пополнения: при каждом внесении средств банк начисляет бонус +10 единиц
3. **BusinessAccount (Бизнес-счет)**
   * Наследуется от базового счета
   * Переопределяет метод снятия: при каждом списании дополнительно взимается комиссия в размере 5 единиц.
